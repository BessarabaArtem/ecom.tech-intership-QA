## Здравствуйте!

В этом репозитории выполненное тестовое задание для Ecom.tech на позицию стажера AQA Python.

## Структура
1. **manual_tests.txt** - Ручные тест-кейсы (Задание 1)
2. **api_tests.py** - Автотесты API на Python + requests (Задание 2)
3. **ui_tests.py** - UI автотест на Playwright (Задание 3)

## Как запустить
Для запуска тестов потребуется ПК с установленным Python(компилятор/интерпретатор языка), если его нет, нужно скачать и установить с официального сайта python.org.

Также понадобятся специальные библиотеки, которые я использовал в коде:
*   **requests** — для API тестов.
*   **playwright** — для UI тестов (открытие браузера).

Перед установкой библиотек проверьте установлен ли Python на вашем устройстве! Введите в терминале `python --version`.
Установите необходимые библиотеки одной командой:
   ```bash
   pip install requests playwright
   ```
Запуск автотестов производится благодаря командам:
```bash
python api_tests.py
python ui_tests.py
```
